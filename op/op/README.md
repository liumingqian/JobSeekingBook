### 系统调用 

系统调用\(System call\)是程序向系统内核请求服务的方式。可以包括硬件相关的服务\(例如，访问硬盘等\)，或者创建新进程，调度其他进程等。系统调用是程序和操作系统之间的重要接口。



### 逻辑地址/物理地址

* 逻辑地址：计算机用户（开发人员）看到的地址。如，创建数组时，操作系统会返回一个逻辑上的连续空间。但逻辑地址其实并不一定是元素存储的真实地址，是操作系统通过地址映射映射成连续的的。
